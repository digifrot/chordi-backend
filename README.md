# Chordi Backend

Backend for Chordi Cloud Save & Song Gallery MVP.

## Setup

1. Copy `.env` and fill in your MongoDB URI
2. Install dependencies:

```bash
npm install
```

3. Run server:

```bash
npm run dev
```

## API Endpoints

- POST /songs
- GET /songs
- GET /songs/:id
- PUT /songs/:id
- DELETE /songs/:id
